# CHANGELOG Generator

**Bounty**: $50 — [claude-builders-bounty #1](https://github.com/claude-builders-bounty/claude-builders-bounty/issues/1)

Auto-generates a structured `CHANGELOG.md` from git history in 3 steps.

## Setup

```bash
chmod +x changelog.sh
```

## Usage

```bash
# Since last git tag
bash changelog.sh

# Since specific tag
bash changelog.sh v1.0.0

# Between two refs
bash changelog.sh v1.0.0 v1.1.0
```

## Categorization Rules

| Prefix | Category |
|--------|----------|
| `feat:` / `add:` / `new:` / `implement:` / `introduce:` / `create:` | **Added** |
| `fix:` / `bug:` / `patch:` / `resolve:` / `hotfix:` / `repair:` / `correct:` | **Fixed** |
| `remove:` / `delete:` / `drop:` / `deprecate:` / `obsolete:` | **Removed** |
| Everything else (chore, docs, refactor, style, test, etc.) | **Changed** |
